# Khanh Linh To Portfolio

Personal developer portfolio for Khanh Linh To, a Computer Science student focused on full-stack development, blockchain certificate verification, AI applications, and AI-IoT monitoring workflows.

## Tech Stack

- React
- TypeScript
- Vite
- lucide-react
- CSS custom properties for dark/light themes

## Features

- Responsive portfolio layout
- Dark/light theme toggle with localStorage persistence
- EN/VI language toggle with localStorage persistence
- Project, skill, education, certification, and contact sections
- CertChain highlighted as the featured project

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

## Notes

- CV download is intentionally disabled until a real CV file is added.
- Certificate evidence is linked through the Google Drive folder shown in the portfolio.
